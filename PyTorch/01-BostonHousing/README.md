# Boston住宅価格予測プロジェクト

## 📋 プロジェクト概要

ボストン地域の住宅価格を予測する回帰モデルを実装したプロジェクトです。PyTorchを使用してニューラルネットワークを構築し、住宅の特徴量から価格を予測します。

## 🎯 目的

- 回帰問題における深層学習の基礎を理解する
- PyTorchの基本的なワークフロー（データ準備、モデル構築、訓練、評価）を習得する
- ニューラルネットワークのハイパーパラメータ調整の経験を積む

## 🛠️ 使用技術

- **フレームワーク**: PyTorch
- **言語**: Python 3.x
- **主要ライブラリ**: NumPy, re

## 📊 データセット

**Boston Housing Dataset**
- **サンプル数**: 506件
- **特徴量数**: 13個（犯罪率、住宅の平均部屋数、税率など）
- **目標変数**: 住宅価格（千ドル単位）
- **分割**: 訓練データ 496件、テストデータ 10件

### 特徴量の説明
1. CRIM: 犯罪率
2. ZN: 25,000平方フィート以上の住宅地の割合
3. INDUS: 非小売業の土地面積の割合
4. CHAS: チャールズ川沿いかどうか（1=Yes, 0=No）
5. NOX: 窒素酸化物濃度
6. RM: 住宅あたりの平均部屋数
7. AGE: 1940年以前に建てられた物件の割合
8. DIS: 5つのボストン雇用センターまでの距離
9. RAD: 高速道路へのアクセス指数
10. TAX: 固定資産税率
11. PTRATIO: 生徒と教師の比率
12. B: 黒人居住者の割合
13. LSTAT: 低所得者の割合

## 🏗️ モデル構造

### ニューラルネットワーク
- **入力層**: 13ユニット（特徴量数）
- **隠れ層**: 100ユニット + ReLU活性化関数
- **出力層**: 1ユニット（価格予測値）

### ハイパーパラメータ
- **損失関数**: MSE（平均二乗誤差）
- **最適化アルゴリズム**: Adam
- **学習率**: 0.01
- **エポック数**: 10,000

## 💡 学習したポイント

### 技術面
1. **PyTorchの基本ワークフロー**: データ準備 → モデル定義 → 損失計算 → 最適化 → 評価
2. **回帰問題の扱い方**: MSE損失関数の使用、連続値の予測
3. **隠れ層の効果**: 単純な線形回帰と比較して、隠れ層を追加することで精度が向上
4. **最適化アルゴリズムの選択**: SGDからAdamに変更することで収束速度が改善
5. **損失スケーリング**: 損失値が大きすぎる場合のスケーリング処理（×0.001）
